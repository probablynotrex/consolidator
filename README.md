# Excel Consolidator

A private, client-side tool to consolidate inventory lists and identify duplicates.

## Features
- **Local Parsing**: Files are parsed entirely in the browser.
- **Privacy First**: No data is sent to any server.
- **Smart Aggregation**: Automatically merges duplicates and sums quantities.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
